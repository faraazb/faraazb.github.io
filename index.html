Projects

DCroSS

Disaster Crowd-sourcing and Support System (DCroSS) is a project built for disaster managers to collect data during time critical disaster situations. It placed third in IITB's e-Yantra Innovation Challenge 2020-21. It uses web scraping and chatbots to aggregate data from various sources and provides a dashboard to consume this data.

Find-a-CoWIN

Find-a-CoWIN is a slot finder for MoHFW's (India) CoWIN vaccination platform. It uses the CoWIN Public API to get the vaccination slots data for any district in any state across India. It has extensive filtering options, light or dark themes and the ability to star or bookmark vaccination centers.

Feedbacker

Feedbacker is a tool that can be used to collect, manage and display product feedbacks on landing pages. Feedbacks or reviews can be collected using a form and embedded into other sites using an iframe, and a dashboard can be used to control which feedbacks have to be displayed.

mycrowbot

mycrowbot is a Telegram bot that can be used by educational institutions to distribute any educational resource to students. The bot can be managed by class, college and super admins who can add and delete resources and also allow or deny access to other students.
